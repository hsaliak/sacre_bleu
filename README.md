# C++ Ninja Google Style Project

This project is set up with:
- **CMake** and **Ninja** build system.
- **Exceptions disabled** (`-fno-exceptions`).
- **Clang-Tidy** integrated with **Google C++ Style**.

## Building

To build the project:

```bash
mkdir build
cd build
cmake -G Ninja ..
ninja
```

## Running

```bash
./main
```
